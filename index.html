Enkripsi-Dekripsi (Vigenère)

Deksripsi:

Web ini dirancang dengan DOM dan mengkombinasikan html, css dan program javascript untuk memanipulasi data. Salah satu contoh pengaplikasiannya adalah dengan menerapkan algoritma untuk enkripsi dan dekripsi dari data berupa teks. Algoritma pengolahan dilakukan lewat javascript yang kemudian memanipulasi element pada html untuk merubah tampilan textbox sesuai dengan output hasil dari enkripsi/dekripsi data.


Referensi untuk algoritma saya ambil dari:

Nayuki.io repl.it Final Project, Hacktiv8: Phase 0
